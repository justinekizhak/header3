Default settings
--------------------------------------

  - `mini-header` is the default header for

    - PHP
    - Shell scripts

  - `file-header` for

    - C
    - C++
    - Clojure
    - Coffee script
    - Erlang
    - Haskell
    - Java
    - Kotlin
    - Python
    - Ruby
    - Rust
    - Scala
    - Scheme
    - Swift

  - `package-header`

    - Emacs Lisp

  - `readme-header`

    - When the file has `.md` extension and has `readme`(case insensitive) in the name.

* You can change the default settings from the customize buffer.
